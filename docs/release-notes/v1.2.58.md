# Storage Redirect X v1.2.58

## 模块更新

### 修复了什么问题

- 改进 scoped FUSE 文件保存、原子替换和同步操作的兼容性，避免部分应用保存或覆盖文件时失败。
- 修复放行实际路径与路径映射视图的可见性和写入权限问题，同时保留公共存储文件的 `media_rw` 元数据。
- 修复 MediaStore 与系统代写公共媒体文件时的归因、保存和存储路由一致性问题。

### 注意事项

- 更新模块后需要重启设备，使 Zygisk、系统媒体进程和挂载状态完整加载新版本。
- 重启后建议确认已配置应用的文件保存位置和公共媒体写入结果符合预期。

## App 更新

### 修复了什么问题

- App 与 WebUI 检查更新时会在主要更新清单源不可用时自动切换到镜像源，提高更新信息获取的可用性。
